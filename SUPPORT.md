# Support

Use [GitHub issues](https://github.com/0John-Hong0/dicron/issues) for reproducible bugs and feature requests.

Before filing an issue, include:

- operating system and app version
- whether you opened a single file, folder, or dropped paths
- a description of the DICOM modality and transfer syntax, if known
- screenshots only when they do not contain private patient information

This project is not a diagnostic medical device. It is intended for local inspection of DICOM images, metadata, and series organization.
