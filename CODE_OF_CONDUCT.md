# Code of Conduct

This project should be a useful, low-drama place to collaborate.

## Expected Behavior

- Be respectful and direct.
- Assume good intent, especially when reviewing code.
- Keep criticism about the work, not the person.
- Be careful with medical-data context. Do not ask contributors to share private studies or patient-identifying information.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Posting private data, credentials, or patient information.
- Repeatedly derailing issues or pull requests after maintainers ask you to stop.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; block users; or take other reasonable steps to keep the project safe and focused.

If you need to report a conduct issue, contact the maintainer privately using the contact information on their GitHub profile. Do not post sensitive details publicly.
